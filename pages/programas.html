PROGRAMAS Y LENGUAJES

PYTHON


Python es un lenguaje de programación que te permite crear programas informáticos de manera fácil y eficiente.





SITIO WEB DE PYTHON


El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.






HTML



Html es un lenguaje de hipertexto. Es un lenguaje que permite definir y darle estructuras a las páginas webs.



SITIO WEB DE HTML


Python es un lenguaje de programación que te permite crear programas informáticos de manera fácil y eficiente. Imagina que es como tener un conjunto de instrucciones claras y precisas que puedes darle a una computadora para que haga lo que tú quieres.




VISUAL STUDIO CODE



Visual Studio Code incluye soporte para la depuración, control integrado de git, resaltado de sintaxis, finalización inteligente de código, fragmentacion y refactorización de código.



ENLACE DE DESCARGA


Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web. Su facilidad de uso, extensibilidad y gratuidad lo convierten en una opción atractiva para programadores de todos los niveles.





C# (C-SHARP)



Es un lenguaje de programación desarrollado por Microsoft, orientado a objetos.



SITIO WEB DE C#


C# (pronunciado "si sharp" en inglés) es un lenguaje de programación moderno, basado en objetos y con seguridad de tipos. Es ampliamente utilizado para crear aplicaciones de escritorio en Windows.




PSEINT



PSeInt es un software libre educativo multiplataforma dirigido a personas que se inician en la programación.



ENLACE DE DESCARGA


PSeInt es una herramienta educativa multiplataforma y gratuita, principalmente dirigida a estudiantes que se inician en el mundo de la programación. Su objetivo principal es facilitar el aprendizaje de los fundamentos de la programación y el desarrollo de la lógica computacional.
